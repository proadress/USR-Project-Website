# Nextra Docs Template 

這是用 nextra 和 tailwind css 開發的網站

## Local Development 本地開發

第一步：(必須)安裝環境
First, run `pnpm i` to install the dependencies.

第二步：執行
Then, run `pnpm dev` to start the development server and visit localhost:3000.

## 資料夾結構

元件都放在components
所有靜態資源都放在public
所有頁面都放在page(主要)

其他盡量不要更改

注意！node_modules不要上傳
